# Accurate Conservative Diffusive Interface Method

![cover image](https://github.com/NaokiHori/CfdToolbox/blob/main/ACDI/cover.jpg)

Two-dimensional ACDI in wall-bounded domains (x: wall-bounded, y: periodic).

## Quick Start

Build and run:

```console
make output
make all
./a.out
```

## Caveats

The primary focus on this package is the implementation of the interface-capturing scheme.
For brevity we assume the density / viscosity ratios are unity.
